Menu bar

The Menu Bar consists of 5 options.

  • File
  • Edit
  • Tools
  • Stressed
  • Help


File

File is used for basic file formatting. Saving(CTRL+S), save as (CTRL+SHIFT+S)opening(CTRL+O), new documents(CTRL+N), printing(CTRL+P), and exporting (CTRL+SHIFT+P). This will host all of your basic needs. These can be access with the stated key shortcuts.


Edit

Edit is used for your writing needs. Undoing (CTRL+Z), redoing(CTRL+Y), cut(CTRL+X), copy(CTRL+C), and paste(CTRL+V). These can be access with the stated key shortcuts.


Tools

Tools host all of your editorial needs. This will consists of bold (CTRL+B), italics (CTRL+I), underlining (CTRL+U), and color coding notes. These can be access with the stated key shortcuts.

This will also have positioning needs. Such as...

LEFT

RIGHT

CENTER


Stressed...

Stressed is a unique option to play a variety of music classical/piano music to calm your nerves. To start the playlist, hit the PLAY button. To stop the music, hit the STOP icon. To pause the music, hit the PAUSE icon. To resume the music, hit the RESUME Icon.
